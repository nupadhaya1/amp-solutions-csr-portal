# Change subscription plan

## Summary

Plan changes can affect pricing, billing, vehicle capacity, and which vehicles are covered.

## CSR remediation steps

1. Verify customer identity.
2. Confirm current plan and desired plan.
3. Explain differences in coverage, price, and vehicle capacity.
4. For downgrade, confirm which vehicle remains covered.
5. Use the plan change action.
6. Review resulting subscription and covered vehicles.
7. Add support note and confirm audit event.

## Portal fields to check

- Current plan
- Target plan
- Subscription status
- Covered vehicles
- Next billing date
- Purchase history if proration is modeled

## Escalate if

- Proration or refund policy is unclear.
- Plan change fails.
- Downgrade would leave an expected vehicle uncovered.

## Customer-safe explanation

I can change the membership plan. If the new plan covers fewer vehicles, we need to confirm which vehicle should remain covered before applying the change.
