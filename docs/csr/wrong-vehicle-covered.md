# Wrong vehicle has subscription coverage

## Summary

The customer may have multiple vehicles but the subscription is assigned to the wrong one.

## CSR remediation steps

1. Verify customer identity.
2. Review all vehicles on the account.
3. Identify which vehicle currently has coverage.
4. Confirm which vehicle should be covered.
5. Use transfer coverage if allowed.
6. Confirm final state and add support note.

## Portal fields to check

- Vehicle list
- Subscription assignment
- Plan capacity
- Audit timeline

## Escalate if

- Plate conflict exists.
- Plan does not allow requested coverage.
- Customer claims both vehicles should be covered under a single-vehicle plan.

## Customer-safe explanation

The membership is currently assigned to a different vehicle. I can move the coverage to the correct vehicle if the plan and account status allow it.
