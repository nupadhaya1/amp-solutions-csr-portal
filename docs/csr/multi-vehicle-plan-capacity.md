# Multi-vehicle plan capacity issue

## Summary

Some plans cover one vehicle while family plans may cover multiple vehicles. Coverage must match plan capacity.

## CSR remediation steps

1. Review current plan and covered vehicle count.
2. Compare covered vehicles against plan capacity.
3. If adding a vehicle exceeds capacity, explain upgrade requirement.
4. If downgrading, ask which vehicle remains covered.
5. Perform plan change or transfer coverage as applicable.
6. Add support note and verify audit timeline.

## Portal fields to check

- Subscription plan
- Plan vehicle capacity
- Vehicle list
- Covered vehicle assignments

## Escalate if

- Vehicle count and plan capacity are inconsistent.
- Customer claims they were promised more vehicles than plan allows.
- UI allows invalid coverage state.

## Customer-safe explanation

Your plan determines how many vehicles can receive membership coverage. If the plan only covers one vehicle, we need to choose which vehicle keeps active coverage.
